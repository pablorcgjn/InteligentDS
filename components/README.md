# Componentes

Esta carpeta contiene todos los componentes reutilizables del sistema de diseño.

## Organización

Los componentes se organizan por categorías:

- **Básicos**: Button, Input, Label, etc.
- **Formularios**: Form, Checkbox, Radio, Select, etc.
- **Navegación**: Navigation, Breadcrumb, Tabs, etc.
- **Layout**: Container, Grid, Stack, etc.
- **Feedback**: Alert, Toast, Modal, etc.

## Convenciones

- Cada componente debe tener su propia carpeta
- Usar tokens de diseño para estilos
- Incluir variantes y estados
- Documentar props y ejemplos de uso

# Tokens de Diseño

Esta carpeta contiene todos los design tokens del sistema.

## Estructura

- **colors/** - Paletas de colores (primarios, secundarios, neutrales, semánticos)
- **typography/** - Tokens de tipografía (familias, tamaños, pesos, alturas de línea)
- **spacing/** - Escala de espaciado y márgenes
- **shadows/** - Sistema de sombras y elevaciones
- **borders/** - Radios de borde y estilos de bordes

## Formato

Los tokens se definen en formato JSON para facilitar su uso en diferentes plataformas.

Ejemplo:
```json
{
  "color": {
    "primary": {
      "value": "#0066CC",
      "type": "color"
    }
  }
}
```

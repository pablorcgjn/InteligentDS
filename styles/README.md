# Estilos Globales

Esta carpeta contiene los estilos globales y configuraciones CSS del sistema.

## Contenido

- **reset.css** - Reset o normalize de estilos
- **global.css** - Estilos globales del sitio
- **utilities.css** - Clases utilitarias
- **themes/** - Variantes de temas (light, dark, etc.)
